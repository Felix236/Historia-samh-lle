Historia och samhälle


Här ska jag skriva lite om historia om Internet och beskriva ett par begrepp och även förklara hur dessa har påverkat samhället

ARPANET var världens första fungerande paketväxlande nätverk och en föregångare till dagens globala internet. Det utvecklades av det amerikanska försvarets utvecklingsavdelning DARPA under kalla kriget och var klart i slutet av 1969. Man använde ARPANET inom militären för att skicka information mellan två källor.


FTP var ett av de tidigaste filöverföringsprotokollet för Internetet. FTP är ett kommandobaserat protokoll för överföring av text och binära datafiler.


Telnet är ett nätverksprotokoll som främst är avsett för textbaserad inloggning på olika datorer, men man kan också använda det för annan textbaserad kommunikation online. Utvecklingen av Telnet började 1969 och var grunden till hur man idag kan skapa konton och sedan logga in för att spara data


SMTP är det vanligaste kommunikationsprotokollet som man använder för att leverera e-post och mail av olika slag. Man började att använda sig av SMTP runt början av 70-talet för att kunna ha ett system som både kunde skicka, ta emot och svara på e-post eller andra meddlanden som skickas online.


HTML ger möjlighet att ange ett dokuments struktur så som rubriker eller styckeindelning. Man kan också ändra metainformation (språk, författare, plats) och hur själva dokumentet ska visas när det öppnas upp. HTML blev släppt och användbart 1993 och har sedan dess fått uppdateringar som ändrar hur språke fungerar samt att de lägger till nya funktioner som man kan använda som du skriver ed HTML


HTTP är det kommunikationsprotokoll som används för att överföra webbsidor på Internet. HTTP och HTML är grunden för hur vårt Internet fungerar idag då de är relativit simpla program som funkar i kombination med varandra.


En webbläsare är en samling av hemsidor och information som lagras på internet. Tex så är Chrome en webbläsare som är skapad av Google där en massa info lagras. Chrome är också kompitabelt med andra appar skapade av Google så du behöver inte ladda ner alla dessa appar utan Chrome länkar dig till en websida som delar information med appen


CSS är ett språk som beskriver stilen för ett strukturerat dokument. Man kan använda CSS för att ändra textstorlek och färg i själva dokumentet
